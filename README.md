# Mis Tareas

Proyecto realizado en Laravel para administrar tareas personales.

## Nombre

David Salazar

## Versiones utilizadas

PHP: 8.5.7

Laravel: 13.15.0

## Cómo ejecutar el proyecto

1. Clonar el repositorio.

```bash
git clone https://github.com/davidcito1212/PruebaTecnica.git
```

2. Entrar a la carpeta del proyecto.

```bash
cd mis-tareas
```

3. Instalar las dependencias.

```bash
composer install
```

4. Verificar que exista el archivo `.env`.

5. Ejecutar las migraciones.

```bash
php artisan migrate
```

6. Iniciar el servidor de Laravel.

```bash
php artisan serve
```

7. Abrir en el navegador:

```text
http://127.0.0.1:8000/tareas
```

## Funcionalidades

* Crear tareas.
* Editar tareas.
* Eliminar tareas.
* Marcar tareas como completadas.
* Asignar prioridad.
* Agregar fecha límite.
